Ringkasan Materi

User Interface adalah Cara/mekanisme User melakukan interaksi dengan program, sehingga user dan program dapat ‘berkomunikasi’ satu sama lain. User interface dapat menerima informasi dari user dan memberikan informasi kepada user (setelah melalui proses yang ada pada program/komputer) untuk kepentingan user. User interface dapat menerima/memberikan informasi kepada user melalui proses interaksi, melalui proses inilah program dan user dapat berinteraksi untuk kepentingan user.

Macam - macam interaksi yang umunya terjadi adalah :

  • 1. Direct Manipulation
  • 2. Menu Selection
  • 3. Form Fill-In
  • 4. Command Language
  • 5. Natural Language
